📊 Google Sheets
La URL que obtenés al deployar el archivo apps-script.gs como Web App
📋 Pasos para conectar:
1. Creá tu Google Sheets con la estructura del archivo estructura-sheets.md
2. En el Sheets, andá a Extensiones → Apps Script
3. Pegá el contenido de apps-script.gs
4. Deploy → New deployment → Web App → Anyone
5. Pegá la URL generada en el campo de arriba
⚠️ Regla de oro: Sheets es siempre la fuente de verdad. Al abrir el portal se leen los datos frescos de Sheets. El portal solo escribe filas nuevas, nunca sobreescribe filas existentes.
